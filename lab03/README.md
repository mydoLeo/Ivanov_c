Ответы на вопросы:
1) Выделяет заданное количество байт памяти без инициализации.
2) calloc выделяет память и заполняет её нулями, а malloc — нет. У calloc два аргумента, у malloc — один.
3) Изменяет размер ранее выделенного блока памяти, сохраняя старые данные.
4) Память остается занятой до завершения программы.
5) Освобождает ранее выделенную память.
6) Возникает ошибка. Это категорически нельзя делать
7) Написать проверку: if (arr == NULL)
8) Статический имеет фиксированный размер и освобождается автоматически, динамический создается во время выполнения и требует ручного управления памятью.
