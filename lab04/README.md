Ответ на вопросы:
1)Файл в C — последовательность байтов, доступная через поток.

2)FILE* — указатель на структуру, описывающую поток.

3)Режимы: r, w, a, r+, w+, a+, +b для бинарных.

4)Текстовый vs бинарный — в текстовом возможна трансляция перевода строки, в бинарном — нет.

5)putc / getc — запись/чтение одного символа.

6)fgets / fputs — чтение/запись строк.

7)fprintf / fscanf — форматированный вывод/ввод.

8)fread / fwrite — побайтовое чтение/запись блоков данных.

9)fseek / ftell — перемещение указателя и получение позиции.

10)feof / ferror — проверка конца файла и ошибок.

11)Закрывать файл важно — иначе возможна потеря данных и утечка ресурсов.
